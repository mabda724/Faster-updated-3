# Sounds Directory

This directory should contain notification sound files for the app.

## Required Files

- `notification.mp3` - Notification sound for new requests (like inDrive)

## How to Add Sounds

1. Place your notification sound file in this directory
2. Name it `notification.mp3`
3. The sound will be played when providers receive new requests

## Recommended Sound

A short, pleasant notification sound (1-2 seconds) that alerts providers to new requests without being annoying.
